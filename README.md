# Dashboard de Vendas E-commerce | Olist

Projeto prático de análise de dados (End-to-End) utilizando dados reais do e-commerce brasileiro (Olist). O objetivo foi simular um desafio real de mercado: sair de arquivos CSV soltos e entregar um painel gerencial automatizado.

<img width="100%" alt="dashboard_final" src="https://github.com/user-attachments/assets/eb0d7ace-e327-491c-8a21-f092869d64c2" />

## O que foi feito
Neste projeto, não utilizei apenas o Power BI. Construí todo o pipeline de dados:

* **Engenharia de Dados (Python):** Script para leitura e tratamento automatizado de múltiplos arquivos CSV, corrigindo tipos de dados e valores nulos.
* **Banco de Dados (SQL):** Criação de um banco SQLite local e estruturação de Views para conectar ao BI (substituindo o uso de tabelas "flat" do Excel).
* **Visualização (Power BI):** Design de interface focado na experiência do usuário (estilo App/SaaS) e Storytelling para responder perguntas de negócio.

## Principais Indicadores (KPIs)
O painel monitora:
* Faturamento Total e Ticket Médio.
* Volumetria de Pedidos por período.
* Mapa de calor de vendas por região (Geolocalização).
